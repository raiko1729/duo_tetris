# DUO TETRIS

2人で1つの盤面を交互に操作する，蹴落とし合いテトリス．

🎮 **プレイURL**: https://duo-tetris.onrender.com

---

## ゲームルール

- P1 → P2 → お邪魔マス出現 → P1 → P2 → お邪魔マス出現... の順でターンが回る
- 自分のターンにミノを1つ操作して着地させる
- ラインを消去すると**自分のスコア**が加算される（相手のスコアとは別）
- 盤面が一番上まで積み上がったらゲームオーバー

### お邪魔マス
- 3ターンごとに，盤面の一番高いブロック（お邪魔マスは除く）以下の空きマスにランダムで2つ出現する
- 通常のブロックと同様に，ラインが揃えば消去できる

### 操作方法
| キー | 動作 |
|------|------|
| ← → | 左右移動 |
| ↑ | 回転 |
| ↓ | ソフトドロップ |
| スペース | ハードドロップ |

- 着地状態で0.5秒操作しないと自動でハードドロップ
- 15秒以内に操作しないとタイムアウト

## ローカルでの起動方法

```bash
# リポジトリをクローン
git clone https://github.com/あなたのユーザー名/duo-tetris.git
cd duo-tetris

# パッケージをインストール
npm install

# サーバー起動
node server.js
```

ブラウザで `http://localhost:3001` を2タブ開いてプレイできます．

## 技術構成

| 項目 | 内容 |
|------|------|
| フロントエンド | HTML / CSS / JavaScript（Canvas） |
| サーバー | Node.js + Socket.io |
| ホスティング | Render（無料プラン） |

### アーキテクチャ
- 盤面の状態はサーバーが管理（Source of Truth）
- クライアントはキー入力の受け付けと描画のみ担当
- ターン交代・ライン消去・お邪魔マス出現はすべてサーバー側で処理
